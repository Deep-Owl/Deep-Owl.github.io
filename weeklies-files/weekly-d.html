Weekly 4

This week I learned more about CSS styling.
Something that I am really interested in is interface design, and I have often wondered how to make flexible designs.
I have wanted to be able to design something that looks different depending on what type of device the viewer is using, and has a certain level of interactivity
So when I started working on my poem project it was important to me to learn how to create an interface that looks different depending on how big the window is. My first step was learning how to make all the content scale to screen.
I created customized tables and used scaling properties such as...
width: 100%;
width: 100vw;

height: 100%;
height: 100vh;

top: 50%;
left: 50%;
transform: translate(-50%, -50%);

display: flex;
















I also learned how to use specific tools like importing text from google fonts
I have installed custom libraries that check my spelling and allow me to color my IDE windows so that I can tell which project I am looking at when I have multiple projects opened
My poem was about symmetry so I used the reflect-x property very liberally.
So far have I been really leaning into using the same tools repetitively but iterating and improving the style of the element each time
I have leaned heavily into styling my own tables, lists, and pre tags

Overall I am very proud of my work so far, in the future I would really like to learn how to make floating objects and make something that can be viewed on a cellular phone.
There is one design that really stuck out to me where there was a floating vector image behind a wall of text and it used the accelerometer built into the phone to float and rotate depending on how you moved the phone.
